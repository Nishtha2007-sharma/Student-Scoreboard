# 🖥️ Student Scoreboard

A terminal-styled student score tracking app built with React + Vite.

## 🔗 Live Demo
[student-scoreboard-nqyd55s0u-nishtha2007-sharmas-projects.vercel.app](https://student-scoreboard-nqyd55s0u-nishtha2007-sharmas-projects.vercel.app)

## ✨ Features
- Add students with scores
- Auto calculates total, passed, and average score
- PASS/FAIL status badges
- Inline score update for each student
- Cyberpunk terminal UI with glowing accents

## 🛠️ Built With
- React
- Vite
- CSS (custom terminal design system)

## 🚀 Run Locally
git clone https://github.com/Nishtha2007-sharma/Student-Scoreboard.git
cd Student-Scoreboard
npm install
npm run dev
